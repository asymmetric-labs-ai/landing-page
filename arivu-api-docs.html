Keystone API — Client APIs v1.0.0

Asymmetric FastAPI Backend with MongoDB support.

Keystone API is a robust backend platform that supports multiple MongoDB databases, enabling scalable, high-performance operations for modern web applications.

Features

  • Database Readiness: Ensures MongoDB databases are ready before startup.
  • Dynamic Route Management: Supports modular routing and exception handling.
  • Logging and Middleware: Provides comprehensive logging and custom middleware support.

Response Encryption

Protected-route responses encrypt data with CryptoV2.1 (secret=device_id, salt=sub from Bearer token). Login / OTP / token responses stay plaintext. Wire payloads carry is_data_encrypted + Base64 ciphertext in data. Toggle via FEATURE_FLAG_V1_CLIENT_WHOLE_PAYLOAD_ENCRYPTION_ENABLED (default: false).

Request Encryption

When whole-payload flag is true, clients send { "encrypted_data": "..." } (CryptoV2.2). Query params stay plaintext. Exclusions: Google/Apple mobile login, health, webhooks.
